# CPP103 – Object-Oriented Programming

**Units:** TBD  
**Target Students:** Computer Engineering students (continuation of CPP102)  
**Language:** Python  
**Version Control:** Git and GitHub

## Course Overview

CPP103 builds on CPP102 by introducing object-oriented programming concepts using Python. Students apply OOP principles — classes, objects, inheritance, encapsulation, and polymorphism — to design and implement structured programs.
