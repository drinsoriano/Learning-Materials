# CPP102 – Programming Logic and Design

**Units:** 2  
**Target Students:** First-year Computer Engineering students  
**Language:** Python  
**Version Control:** Git and GitHub (integrated throughout the course)

## Course Overview

CPP102 introduces programming fundamentals using Python. Students learn algorithmic thinking, flowcharting, pseudocode, and Python programming while building version control habits through Git and GitHub.

## Folder Guide

| Folder | Contents |
|--------|----------|
| OBTLP/ | Outcomes-Based Teaching and Learning Plan (official course document) |
| Week01/ | Introduction to programming, algorithms, flowcharts, Git setup |
| Week02/ | Python basics — variables, data types, operators |
| Assessments/ | Quizzes, lab exercises, project rubrics, GitHub submission guidelines |
| References/ | Textbooks, online resources, Python and Git documentation links |
| Templates/ | Lab activity sheets, rubric templates, README templates for student projects |

## Key Topics

- Algorithm design, pseudocode, flowcharts
- Python: variables, data types, operators, conditionals, loops, functions
- Python: lists, strings, dictionaries, file handling
- Git and GitHub: basic version control workflow
- Final project: Python-based program submitted via GitHub repository
